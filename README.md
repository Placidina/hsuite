# HSuite

HSuite is a HTTP Protocol brute force.

Works on Python versions:

- 2.6.x and greater
- 2.7.x and greater
- 3.4.x and greater
- 3.5.x and greater
- 3.6.x and greater
- 3.7.x and greater
- 3.8.x and greater

## Tools

- [S3Dump](docs/s3dump.md)

## Install

- ***pip:*** `pip install .`
- ***setuptools:*** `python setup.py install`
